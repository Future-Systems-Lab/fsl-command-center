# FSL HypnoNeuro — Promotional Demo Script

**Format:** Screen recording, no voiceover required (on-screen text tells the story)
**Target length:** 4–6 minutes
**Resolution:** 1920×1080, dark mode, Chrome, MetaMask installed
**URL:** https://hypnoneuro-frontend.vercel.app

---

## Pre-Recording Checklist

- [ ] MetaMask connected to Sepolia testnet with test ETH
- [ ] Wallet has no prior sessions (fresh demo wallet preferred)
- [ ] Chrome DevTools closed, browser at 100% zoom
- [ ] Daily.co DAILY_API_KEY set on VPS (avatar session will show iframe)
- [ ] Screen recorder running at 60fps
- [ ] Slow down mouse movements — deliberate, not rushed

---
